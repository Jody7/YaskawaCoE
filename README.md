# YaskawaCoE
Canopen over Ethercat stack is developed specifically for Yaskawa drive control.

Dependencies
------------

 * CMake 2.8.0 or later
 * Python 3.5.1 or later
 * Xenomai 3.0.5 on linux-4.9.38
