# YaskawaCoE
Canopen over Ethercat stack is developed specifically for Yaskawa drive control.

Dependencies
------------

 * CMake 2.8.0 or later
 * Xilinx SDK 2017.3

Building
--------
  * To build & install the project, run the following from this project's root
directory:
    * mkdir build
    * cd build
    * cmake ..
    * make
    * make install
